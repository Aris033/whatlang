# whatlang

Base project created with Vite + React + TypeScript for an English learning app focused on vocabulary and phrase translation.

## Install dependencies

```bash
npm install
```

## Start the project

```bash
npm run dev
```

## Initial structure

```text
src/
  components/
  lib/
  pages/
  services/
  types/
```

## Current pages

- `Home`
- `Login`
- `Practice`
- `Mistakes`
- `Stats`

## Notes

- Simple navigation is handled inside `App.tsx`.
- No Supabase, authentication or complex business logic has been added yet.
